Gletovanje

Gletovanje je postupak finog ravnjanja unutrasnjih ili spoljnih zidnih povrsina pomocu glet hoble i spahtle, kao priprema za krecenje. Koristi se kada neravnine nisu vece od tri milimetra, posto se glet skuplja prilikom susenja.
